プロジェクト: ブラックジャックゲーム

概要

職業訓練校の卒業制作プロジェクトとして開発した、シンプルながらも基本的な機能が揃ったブラックジャックゲームです。4つのゲームが遊べるアプリケーションの一部として作成しました。

機能

  • ディーラーとプレイヤーの対戦
  • ベット機能
  • 履歴表示
  • ヒット（カードを引く）
  • スタンド（勝負する）

使用技術

  • Java (バックエンド、ゲームロジック、サーブレット)
  • JSP (フロントエンド、動的なWebページの生成)
  • HTML (フロントエンド、ページ構造)
  • CSS (フロントエンド、デザイン・レイアウト)
  • Apache Tomcat (Webサーバー)
  • Eclipse (IDE)

このプロジェクトで発揮した能力と得られた経験

  • JavaとWeb技術（JSP/Servlet）を用いたWebアプリケーションの構築プロセスを一通り経験し、企画から設計、実装、テストまでの一連のWebアプリケーション開発フローを理解し、実行することができます。
  • ブラックジャックの複雑なゲームロジック（カード配布、手札計算、勝敗判定など）をJavaで実装することで、複雑な要件を論理的に分解し、堅牢で効率的なビジネスロジックを設計・実装する能力を培いました。
  • ユーザーのゲーム状態や履歴、そしてアプリ全体で共有される所持コインをセッション・アプリケーションスコープで管理することで、Webアプリケーションにおける状態管理の重要性を理解し、状況に応じた適切なデータ保持・連携の実装が可能です。
  • HTMLとCSSを使い、ユーザーが利用しやすいUIデザインと、視覚的に魅力的なページレイアウトを実現しました。ユーザー体験を考慮したフロントエンドの設計・実装に貢献できます。
  • MVCモデルを意識してコードを構成し、役割分担されたコードの保守性や再利用性を実践しました。これにより、可読性が高く、拡張性のあるコードを書くことを意識しており、チーム開発においても品質向上に貢献できます。

動作イメージ

スキル

  • Java:
    職業訓練校でのメイン言語として、ブラックジャックゲームの複雑なゲームロジックやバックエンド処理の設計・実装を担当しました。オブジェクト指向プログラミングに基づいた堅牢なシステム構築が可能です。
  • HTML:
    ポートフォリオサイトおよびブラックジャックゲームのWebページ構造の設計とマークアップを遂行。セマンティックなHTML記述により、可読性と保守性の高いコードを作成できます。
  • CSS:
    ポートフォリオサイトおよびゲームのレイアウトとデザイン実装を担当。Flexboxを活用した柔軟な配置や、モバイル環境にも対応するレスポンシブデザインの実装経験があります。
  • JSP:
    Java Servletと連携し、動的なWebページの生成とデータ表示に利用。バックエンドからの情報を効果的にユーザーインターフェースに反映させるスキルがあります。
  • Servlet:
    Webリクエストの処理、ビジネスロジックの呼び出し、JSPへのフォワードなど、Webアプリケーションの主要なコントローラ部分の実装を経験。セッション・アプリケーションスコープを用いた状態管理も適切に行えます。
  • Apache Tomcat:
    開発したWebアプリケーションのデプロイと基本的なサーバー環境の構築・運用を経験。開発したアプリケーションを実際に動作させるための環境構築が可能です。
  • Eclipse:
    開発IDEとして日常的に利用。効率的なコーディング、デバッグ、Git連携によるバージョン管理を実践しており、開発フローに沿った作業をスムーズに行えます。

今後学習したい技術

  • RDBMS (例: MySQL/PostgreSQL): データベース設計・操作の知識を深め、よりデータ駆動型のアプリケーション開発に貢献するため、SQLの学習と実践を進めています。
  • Git / GitHub: バージョン管理システムをチーム開発で活用し、より効率的で協調的な開発フローを実践していきたいです。特にPull RequestやIssue管理についても習熟を目指しています。
